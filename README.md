# Justin Garey Personal Website & Portfolio

I wanted a place to list my projects, write blogs, and host my web games. This is it!

## Creation

```bash
npm create vite@latest JustinGareyWebsite -- --template react-ts
```

## Usage

Run Development Server
```bash
npm run dev
```

Host Development Server to Network
```bash
npm run dev -- --host
```

## Installed Packages

[Tailwind](https://tailwindcss.com/docs/installation/using-vite):
```bash
npm install tailwindcss @tailwindcss/vite
```

Browser Router 
```
npm install react-router-dom
```

React Markdown
```bash
npm install react-markdown
```

GitHub Flavored Markdown
```bash
npm install react-gfm
```
